html5 meter element

Note:- Edit the code below for fun! to tweak the meter gauge above.

<meter max= min= value= high= low= optimum=></meter>
Case 1 - No attributes <meter></meter>
Case 2 - value < max (default: min=0, max=1) <meter value=".5"></meter>
Case 3 - value = max (default: min=0, max=1) <meter value="1"></meter>
Case 4 - value > max (default: min=0, max=1) <meter value="5"></meter>
(if value > max then value = max)
Case 5 - value < min (default: min=0, max=1) <meter value="-.5"></meter>
(if value < min then value = min)
Case 6 - value = min (default: min=0, max=1) <meter value="0"></meter>
Case 7 - value > min (default: min=0, max=1) <meter value=".5"></meter>
Case 8 - value < high (default: min=0, max=1) <meter value=".5" high=".8"></meter>
Case 9 - value = high (default: min=0, max=1) <meter value=".8" high=".8"></meter>
Case 10 - value > high (default: min=0, max=1) <meter value=".9" high=".8"></meter>
(if value > high then notice how the color changes)
Case 11 - value < low (default: min=0, max=1) <meter value=".15" low=".25"></meter>
(if value < low then notice how the color changes)
Case 12 - value = low (default: min=0, max=1) <meter value=".25" low=".25"></meter>
Case 13 - value > low (default: min=0, max=1) <meter value=".5" low=".25"></meter>
Case 14 - optimum < low < high (default: min=0, max=1) <meter low=".25" optimum=".15" high=".75" value=".5"></meter>
(if optimum < low then notice how the color changes)
Case 15 - low < optimum < high (default: min=0, max=1) <meter low=".25" optimum=".5" high=".75" value=".5"></meter>
Case 16 - low < high < optimum (default: min=0, max=1) <meter low=".25" optimum=".85" high=".75" value=".5"></meter>
(if optimum > high then notice how the color changes)
Case 17 - value < low < high < optimum (default: min=0, max=1) <meter low=".25" optimum=".8" high=".75" value=".2"></meter>
(if value < low and optimum > high then notice how the color changes)
Case 18 - value > high > low > optimum (default: min=0, max=1) <meter low=".25" optimum=".2" high=".75" value=".8"></meter>
(if value > high and optimum < low then notice how the color changes)
